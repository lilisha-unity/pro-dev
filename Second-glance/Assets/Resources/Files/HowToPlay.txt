1. Select Start to begin.
2. The screen shows an image.
3. You have a few seconds to memorize the image before it changes.
4. If the image has appeared before, select the card to earn points.
5. If the image is new, do not select it. Wait for the next card.
6. You earn points for selecting repeated images.
7. You lose points for selecting new images or missing repeated images.
8. As the game continues, the card changes more quickly, making it harder to remember previous images.
9. You start with three lives. The game ends when you run out of lives.

You can mute/unmute the speaker by pressing the space bar.
